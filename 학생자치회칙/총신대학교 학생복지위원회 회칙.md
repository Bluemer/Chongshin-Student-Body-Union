# 학생복지위원회
## 제1장 총칙 <a id="chapter-1"></a>
### 제1조 <a id="article-1"></a>
(명칭) 본회는 ‘총신 대학교 학생 복지 위원회’라 칭한다. (이하 ‘본회’라 칭한다)
### 제2조  <a id="article-2"></a>
(목적) 본회는 기독교적 세계관을 바탕으로, 학원 내의 업체 및 기타 기관들로 부터, 자본주의 이데올로기에 의한 개인주의, 이기주의, 경쟁주의를 지양하게 하고, 하나님의 절대적인 공의의 실현과 학우들의 협동적인 복지생활을 위하여, 봉사하고 헌신함을 그 목적으로 한다.
### 제3조  <a id="article-3"></a>
(회원선거)
#### 제1항 <a id="article-3-clause-1"></a>
학생복지위원회는 본회 회원들의 직접선거로 구성된다. 
#### 제2항 <a id="article-3-clause-2"></a>
추천받은 회원은 위원장의 결정과 대의원총회의 인준을 얻어 임명한다.
### 제4조  <a id="article-4"></a>
(회원의 자격과 조건)
#### 제1항 <a id="article-4-clause-1"></a>
정회원: 본회의 정회원은 총신대학교 재학 중인 학생으로 한다. 위원장과 부위 원장의 동의가 있는 자만이 가능하다. 
#### 제2항 <a id="article-4-clause-2"></a>
준회원: 본회의 준회원은 재학생만 포함한다. 신입생, 편입생, 복학생의 해당 학기도 가능하다. 
### 제5조  <a id="article-5"></a>
(회원의 권리와 의무)
#### 제1항 <a id="article-5-clause-1"></a>
정회원: 선거권, 피선거권을 가지며, 본회의 모든 자치 활동에 참여 할 권리와 의무를 가진다. 
#### 제2항 <a id="article-5-clause-2"></a>
준회원: 시설 및 기구들의 관리의 의무와 사용의 권한을 지닌다. 단, 본인의 담당 시간일 때에만 적용한다.

## 제2장 총회 및 회의 <a id="chapter-2"></a>
### 제6조  <a id="article-6"></a>
(구성)
#### 제1항 <a id="article-6-clause-1"></a>
제4조 “회원의 자격과 조건” 중 ‘정위원’에게만 적용한다. 
#### 제2항 <a id="article-6-clause-2"></a>
‘준위원’은 회의에 참관은 할 수 있으나, 선거권, 피선거권 및 의결권은 없다. 
### 제7조  <a id="article-7"></a>
(지위) 본회의 최고 기구이다.
### 제8조  <a id="article-8"></a>
(역할)
#### 제1항 <a id="article-8-clause-1"></a>
본회의 회칙을 제청하고 필요에 따라 개정한다. 
#### 제2항 <a id="article-8-clause-2"></a>
해당 년도 기타 중요한 안건에 대하여 상정하고 의결한다.
### 제9조  <a id="article-9"></a>
(소집) 회의는 ‘정기회의’, ‘임시회의’로 구분한다.
#### 제1항 <a id="article-8-clause-1"></a>
정기회의 : 매 주 한 번 소집한다.
#### 제2항 <a id="article-8-clause-2"></a>
임시회의 : 위원장의 필요와 재적인원 1/2의 동의 시, 소집할 수 있다.
### 제10조 <a id="article-10"></a>
(의결) 재적인원 1/2 이상의 출석과, 출석인원 1/2이상의 동의 시 의결할 수 있다.

## 제3장 위원장 <a id="chapter-3"></a>
### 제11조 <a id="article-11"></a>
(지위) 본회를 대내외적으로 대표하며 ‘정기회의, 임시회의’ 의장이 된다.
### 제12조 <a id="article-12"></a>
(권한과 의무)
#### 제1항 <a id="article-12-clause-1"></a>
본회의 업무들을 적극적으로 관리할 권한을 지닌다. 
#### 제2항 <a id="article-12-clause-2"></a>
본회의 회원들을 적극적으로 옹호할 의무를 지닌다. 
#### 제3항 <a id="article-12-clause-3"></a>
부위원장 및 각 임원들을 임명할 수 있으며, 회원들의 동의를 얻는다. 
#### 제4항 <a id="article-12-clause-4"></a>
임시총회 및 임시회의를 소집할 수 있다.
### 제13조 <a id="article-13"></a>
(선출)
#### 제1항 <a id="article-13-clause-1"></a>
본회의 회원으로서 1년 이상 가입한 자에 한한다. 
#### 제2항 <a id="article-13-clause-2"></a>
본교의 학생으로서 3학년 이상인 자에 한한다.
#### 제3항 <a id="article-13-clause-3"></a>
본회를 적극적이고 능률적으로 이끌어 갈 수 있는 자에 한한다. 
#### 제4항 <a id="article-13-clause-4"></a>
회원 2/3이상 출석과, 출석인원 2/3 이상의 의결로 선출한다. 
#### 제5항 <a id="article-13-clause-5"></a>
위원장은 학복위회에서 무기명 투표로 선거하며 상임위원회의 인준에 의해 임명 된다.
### 제14조 <a id="article-14"></a>
(임기)
#### 제1항 <a id="article-14-clause-1"></a>
위원장 및 회원들의 임기는 매년 12월 1일부터 익년 11월말까지로 한다. 
#### 제2항 <a id="article-14-clause-2"></a>
회원들의 임기 내에 사퇴하게 될 때에는, 임시총회를 소집할 수 있다.

## 제4장 재정 <a id="chapter-4"></a>
### 제15조 <a id="article-15"></a>
(경비)
#### 제1항 <a id="article-15-clause-1"></a>
복지기금 : 매점, 서점, 복사실, 미용실, 자판기 등의 복지기금으로 한다. 
#### 제2항 <a id="article-15-clause-2"></a>
자체기금 : 사물함 및 기타 판매 수익금으로 한다.
### 제16조 <a id="article-16"></a>
(예산 및 결산)
#### 제1항 <a id="article-16-clause-1"></a>
복지기금 : 총신대학교 기획과로부터 예산안 및 내역을 심의 받는다. 총신대학교 경리과로부터 결산안 및 영수증을 감사 받는다. 
#### 제2항 <a id="article-16-clause-2"></a>
자체기금 : 총신대학교 대의원회로부터 예산안 및 내역을 심의 받는다. 총신대학교 대의원회로부터 결산안 및 영수증을 감사 받는다.
### 제17조 <a id="article-17"></a>
(회계년도) 본 회의 회계 년도는 매년 1월 1일부터 동년 12월 말 까지 1년으로 한다

## 제5장 부칙 <a id="chapter-5"></a>
### 제18조 <a id="article-18"></a>
(적용) 본 회칙은 공포와 함께 즉시 효력을 발생한다.
