# 신학과
## 제1장 총칙 <a id="chapter-1"></a>
### 제1조 <a id="article-1"></a>
(명칭) 본회는 신학과 학생회라 칭한다(이하 ‘본회’라 한다).
### 제2조  <a id="article-2"></a>
(강령) 본회는 총신대학 총학생회의 수권범위 내에서 자율적으로 운영한다.
### 제3조  <a id="article-3"></a>
(목적) 본회는 개혁주의 신앙에 입각하여 올바른 신학정립과 창조적인 학문 탐구를 통하여 교회와 국가에 이바지할 유능한 인재를 길러내고 회원 상호간에 친목을 도 모한다. 궁극적으로는 하나님 나라 건설에 이바지 하는 것을 목적으로 한다.
### 제4조  <a id="article-4"></a>
(회원의 자격) 본회의 회원은 총신대학 신학과의 재학생으로 한다. (단, 휴학 상태 에 있는 자는 회원의 자격이 정지된다.)
### 제5조  <a id="article-5"></a>
(회원의 권리와 의무) 본회의 회원은 다음과 같은 권리와 의무를 가진다.
#### 제1항 <a id="article-5-clause-1"></a>
본회의 회원은 선거권 및 피선거권을 가지며 본회의 활동에 참여할 권리와 의 무를 가진다. 
#### 제2항 <a id="article-5-clause-2"></a>
본회의 회원은 본회의 제반활동에 대하여 알 권리를 가진다. 
#### 제3항 <a id="article-5-clause-3"></a>
본회의 회원은 회칙을 준수하며 회비를 납부할 의무를 가진다.
### 제6조   <a id="article-6"></a>
(구성) 본회는 제 3조의 목적을 달성하기 위하여 학생총회, 대의원총회, 집행부로 구성한다.

## 제2장 학생총회 <a id="chapter-2"></a>
### 제7조   <a id="article-7"></a>
(지위) 학생총회는 본 회의 최고 기구이다.
### 제8조 <a id="article-8"></a>
(구성) 학생총회는 그 회에 포함된 모든 회원으로 구성한다.
### 제9조 <a id="article-9"></a>
(권한)
#### 제1항 <a id="article-9-clause-1"></a>
학생총회는 본회의 운영전반 (결산 시 예산편성 및 집행에 관한 동의권 포함)에 관한 중요한 사항을 보고 받는다. 
#### 제2항 <a id="article-9-clause-2"></a>
학생총회는 과 학생회의 존립 및 과 학생에 관한 중대한 사항을 의결한다. 
#### 제3항 <a id="article-9-clause-3"></a>
학생총회는 학생회장단에 대한 탄핵소추권을 갖는다. 
#### 제4항 <a id="article-9-clause-4"></a>
학생총회는 회칙개정에 대한 발의, 동의 및 의결권을 갖는다.
### 제10조 <a id="article-10"></a>
(소집) 학생회는 정기총회와 임시총회를 둔다.
#### 제1항 <a id="article-10-clause-1"></a>
정기총회는 매학기 개강 후 20일 이내 학생회장이 소집한다. 
#### 제2항 <a id="article-10-clause-2"></a>
임시총회는 운영위원회의 요구나 학생총회의 요구 및 회원 50인 이상의 요구가 있을 경우, 대의원 2인 이상의 요구 시 학생회장이 소집한다.(단, 긴급한 상황에 관한 개최는 학생회장에 의해 소집될 수 있다.) 
#### 제3항 <a id="article-10-clause-3"></a>
정기총회의 소집은 소집 3일전에 공고해야 한다.(단. 임시총회는 예외로 할 수 있다.) 
### 제11조 <a id="article-11"></a>
(의장) 학생총회의 의장은 학생회장이 겸하며, 학생회장의 궐위 시에는 부회장이 이를 대행한다.
### 제12조 <a id="article-12"></a>
(의결) 학생총회의 모든 의결은 재적회원의 100명이상의 참석과 참석회원의 과반 수의 찬성으로 의결한다.

## 제3장 대의원총회 <a id="chapter-3"></a>
### 제13조 <a id="article-13"></a>
(지위) 대의원총회는 본회의 최고 의결기구이다.
### 제14조 <a id="article-14"></a>
(구성)
#### 제1항 <a id="article-14-clause-1"></a>
대의원총회는 본회의 대의원 전원으로 구성한다. 
#### 제2항 <a id="article-14-clause-2"></a>
대의원은 본교 전체학과의 각 학년별 1인으로 한다. 단, 1개 학년의 재학인원이 60인을 초과할 경우에 대표 1인을 추가로 선출한다.
### 제15조 <a id="article-15"></a>
(임기) 대의원의 임기(의장단 제외)는 매년 12월 1일부터 다음 년도 11월 31일까 지로 한다.
### 제16조 <a id="article-16"></a>
(의장)
#### 제1항 <a id="article-16-clause-1"></a>
의장은 상임위원회에서 추천되어 대의원총회의 직접선거로 선출된다. 
#### 제2항 <a id="article-16-clause-2"></a>
의장은 대의원총회 재적 대의원의 과반수 득표로 당선한다.
### 제17조 <a id="article-17"></a>
(업무 및 권한) 대의원회는 다음과 같은 업무 및 권한을 가진다.
#### 제1항 <a id="article-17-clause-1"></a>
학생회 소집 요구권. 
#### 제2항 <a id="article-17-clause-2"></a>
집행부 및 분과 예산의 심의 및 회계 감사권. 
#### 제3항 <a id="article-17-clause-3"></a>
학생회장 탄핵발의권. 
#### 제4항 <a id="article-17-clause-4"></a>
학생회장이 제출한 사업계획 및 보고에 관한 감사권. 
#### 제5항 <a id="article-17-clause-5"></a>
회칙개정 발의권. 
#### 제6항 <a id="article-17-clause-6"></a>
학생회장이 지명한 임원의 인준권. 
#### 제7항 <a id="article-17-clause-7"></a>
학생회장 탄핵 시 잠정적으로 권한을 대행한다.
### 제18조 <a id="article-18"></a>
(정기회) 정기회(예결산 감사)는 3월, 6월, 9월, 11월에 의장이 소집한다.

## 제4장 학생회장 및 부학생회장 <a id="chapter-4"></a>
### 제19조 <a id="article-19"></a>
(지위)
#### 제1항 <a id="article-19-clause-1"></a>
학생회장은 본회를 대표하며 총학생회 운영위원이 된다. 
#### 제2항 <a id="article-19-clause-2"></a>
학생회장은 학생회의 의장이 되며 집행부의 임원을 임명 및 해임할 수 있는 권 한을 갖는다. 
#### 제3항 <a id="article-19-clause-3"></a>
부학생회장은 회장을 보좌하며 회장의 궐위 시 그 직책을 대행한다.
### 제20조 <a id="article-20"></a>
(임기) 학생회장과 부학생회장의 임기는 매년 12월 1일부터 다음 년도 11월 30일 까지 한다. 단, 비상특별위원회 발족 시 학생회장의 임기는 다음 해 진행되는 재투 표 전까지 연장된다.
### 제21조 <a id="article-21"></a>
(신분보장) 학생회장, 부학생회장은 전체학우를 대상으로 하는 불신임결의에 의하지 아니하고는 어떠한 이유로도 그 직위에서 해임되지 아니한다. 단, 투표율은 60% 이상이며 2/3이상의 찬성이 있어야 한다.
### 제22조 <a id="article-22"></a>
(임무 및 권한) 학생회장의 임무 및 권한을 갖는다.
#### 제1항 <a id="article-22-clause-1"></a>
모든 집행부 임원을 임명한다. 
#### 제2항 <a id="article-22-clause-2"></a>
본회의 결산 보고서를 대의원총회 감사 후 학생총회에서 보고한다. 
#### 제3항 <a id="article-22-clause-3"></a>
학생회의 소집 요구권을 가진다. 
#### 제4항 <a id="article-22-clause-4"></a>
집행부 및 분과의 전반적인 사항을 검토 심의 조정한다. 
#### 제5항 <a id="article-22-clause-5"></a>
학생회칙 개정 발의권을 가진다. 
#### 제6항 <a id="article-22-clause-6"></a>
대의원회에 출석, 발언할 수 있다. 
#### 제7항 <a id="article-22-clause-7"></a>
기타 학생회에 관한 중요 사항을 집행한다. 
#### 제8항 <a id="article-22-clause-8"></a>
총회에 안건을 상정할 수 있다.
### 제23조 <a id="article-23"></a>
(선출) 학생회장, 부학생회장은 학생자치단체 선거규약에 의하여 선출한다.

## 제5장 집행부 <a id="chapter-5"></a>
### 제24조 <a id="article-24"></a>
(지위) 집행부는 본회의 최고 집행기구이다.
### 제25조 <a id="article-25"></a>
(구성) 집행부는 학생회장, 부학생회장 및 각 부 부장으로 구성한다.
### 제26조 <a id="article-26"></a>
(업무 및 권한)
#### 제1항 <a id="article-26-clause-1"></a>
본회의 제반활동에 대한 계획을 수립하고 그 수행을 위한 예산, 결산을 편성한 다. 
#### 제2항 <a id="article-26-clause-2"></a>
대의원회에서 의결된 사항을 집행한다. 
#### 제3항 <a id="article-26-clause-3"></a>
집행부는 제반활동을 위해 다음과 같은 부서를 둔다.
##### 1호 <a id="article-26-clause-3-item-1"></a>
총무부.
##### 2호 <a id="article-26-clause-3-item-2"></a>
회계부.
##### 3호 <a id="article-26-clause-3-item-3"></a>
학술부.
##### 4호 <a id="article-26-clause-3-item-4"></a>
영성부.
##### 5호 <a id="article-26-clause-3-item-5"></a>
홍보부.
#### 제4항 <a id="article-26-clause-4"></a>
단 전항 이외의 부서가 필요한 경우에는 학생회장의 발의와 상임위원회의 승 인을 거쳐 추가로 부서를 설치할 수 있다.

## 제6장 재정 <a id="chapter-6"></a>
### 제27조 <a id="article-27"></a>
(재정)
#### 제1항 <a id="article-27-clause-1"></a>
본회의 재정은 총학생회비(학생경비)와 그 이자, 지원금으로 한다. 
#### 제2항 <a id="article-27-clause-2"></a>
본회의 재정은 본회 활동 이외의 다른 목적에 사용할 수 없다.
### 제28조 <a id="article-28"></a>
(회계연도) 본회의 회계 연도는 매년 12월 1일부터 다음 년도 11월 말까지의 1년 으로 한다. (총학생회의 임기와 동일)
### 제29조 <a id="article-29"></a>
(예산) 학생회비 징수는 학교 당국에 위임하고 예산은 집행부 및 운영위원회에서 심의조정한 뒤 대의원총회 의장의 인준에 의해 행한다.

## 제7장 선거 <a id="chapter-7"></a>
### 제30조 <a id="article-30"></a>
(선거권) 선거권은 본회의 재학생으로 당해 학기 등록을 필한 자로 한다.
### 제31조 <a id="article-31"></a>
(선거 방법) 본회의 선거는 보통, 직접, 비밀, 평등선거로 한다.
### 제32조 <a id="article-32"></a>
(선거 시기) 총학생회장단, 각 기구 및 학회장 그리고 각 과 대의원 등의 선거는 임기만료 10일 이전에 실시한다. 단, 1학년 대의원은 신입생 오리엔테이션 기간에 선출한다.
### 제33조 <a id="article-33"></a>
(임기)
#### 제1항 <a id="article-33-clause-1"></a>
총학생회 내 모든 임원의 임기는 매년 12월1일부터 다음 년도 11월말까지로 한 다. 
#### 제2항 <a id="article-33-clause-2"></a>
대의원총회 의장단의 임기는 매년 7월 1일부터 다음 년도 6월말까지로 한다.
### 제34조 <a id="article-34"></a>
(각 과 학생회장 선거)
#### 제1항 <a id="article-34-clause-1"></a>
(피선거권)
##### 1호 <a id="article-34-clause-1-item-1"></a>
4학기이상 등록을 필하고 평점2.5이상인자.
##### 2호 <a id="article-34-clause-1-item-2"></a>
각 과 회원 20인 이상의 추천을 받은 자.
##### 3호 <a id="article-34-clause-1-item-3"></a>
집행부 임원, 각 과학생회 임원, 대의원은 이직을 겸직 할 수 없다.
#### 제2항 제1항 <a id="article-34-clause-2"></a>
선거는 해당학과 전 회원의 직접선거에 의하여 선출되며 세부사항은 선거관리위 원회에 일임한다.

## 제8장 비상특별위원회 <a id="chapter-8"></a>
### 제35조 <a id="article-35"></a>
(직위) 비상특별위원회는 본회의 임시적인 최고 집행기구이다.
### 제36조 <a id="article-36"></a>
(구성)
#### 제1항 <a id="article-36-clause-1"></a>
비상특별위원회가 구성될 경우 현 총학생회장이 대표성을 가지고 비상특별위 원회의 위원장이 된다.
#### 제2항 <a id="article-36-clause-2"></a>
비상특별위원회의 집행부는 비상특별위원회 위원장, 비상특별위원회 부위원장 및 각 집행부장으로 구성된다.
#### 제3항 <a id="article-36-clause-3"></a>
비상특별위원회 위원장은 집행부를 구성할 수 있으며, 구성원은 대의원총회의 인준을 받아 활동한다.
### 제37조 <a id="article-37"></a>
(임기) 비상특별위원회와 비상특별위원장은 본회가 구성된 다음 학기, 새로운 학 생회장이 선출될 때 까지 운영되고 활동한다.
### 제38조 <a id="article-38"></a>
(업무 및 권한)
#### 제1항 <a id="article-38-clause-1"></a>
비상특별위원회는 대의원총회의 인준을 받아 집행부 소관 업무를 집행한다.
#### 제2항 <a id="article-38-clause-2"></a>
비상특별위원회는 운영위원회에서 결정된 사안을 집행한다.
#### 제3항 <a id="article-38-clause-3"></a>
비상특별위원장은 대의원총회, 상임위원회, 운영위원회에서 요구 시 출석하여 발언하여야 한다.
#### 제4항 <a id="article-38-clause-4"></a>
비상특별위원회는 제반활동을 원활히 집행하기 위하여 다음과 같은 부서를 둔다.
##### 1호 <a id="article-38-clause-4-item-1"></a>
사무부 : 집행부의 업무를 총괄적으로 진행하며, 행정 및 행사진행을 담당한다.
##### 2호 <a id="article-38-clause-4-item-2"></a>
정책부 : 대·내외적인 정책을 집행하며, 총학생회 이하 기관의 시스템 구축 작업을 한다.
##### 3호 <a id="article-38-clause-4-item-3"></a>
홍보부 : 대·내외적인 홍보를 기획, 담당한다.
##### 4호 <a id="article-38-clause-4-item-4"></a>
재정부 : 경리 및 회계를 담당한다.
##### 5호 <a id="article-38-clause-4-item-5"></a>
영성부 : 각종 예배와 선교에 관한 대·내외적인 업무를 담당한다.
#### 제5항 <a id="article-38-clause-5"></a>
전항 이외의 부서가 필요한 경우에는 비상특별위원회 위원장의 발의와 상임 위원회의 승인을 거쳐 추가로 부서를 설치할 수 있다. 6
#### 제6항 <a id="article-38-clause-6"></a>
비상특별위원회의 위원장은 사업계획안, 사업보고서, 예산편성과 사업평가서, 결 산서를 작성하여 대의원 총회와 운영위원회에 보고한다.

## 제9장 회칙개정 <a id="chapter-9"></a>
### 제39조 <a id="article-39"></a>
(발의) 회칙 개정의 발의 요건은 다음과 같다.
#### 제1항 <a id="article-39-clause-1"></a>
재적 대의원 과반수이상의 발의.
#### 제2항 <a id="article-39-clause-2"></a>
학생회장의 발의. 
#### 제3항 <a id="article-39-clause-3"></a>
회원 2/3인 이상의 발의.
### 제40조 <a id="article-40"></a>
(의결)
#### 제1항 <a id="article-40-clause-1"></a>
회칙개정 발의 시 학생회장은 이를 즉시 공고하고 공고일로부터 20일 이내에 대의원회에서 재적 대의원 2/3이상의 찬성으로 의결하여 학생회장이 이를 3일 이내에 공포한다.

## 제10장 부칙 <a id="chapter-10"></a>
### 제41조 <a id="article-41"></a>
본 회칙은 공포와 함께 즉시 효력을 발생한다.
